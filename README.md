## 🎯 Overview

This project analyzes sales data for FNP to uncover trends, patterns, and opportunities for business growth. The interactive Excel dashboard provides a comprehensive view of sales performance across different dimensions including products, customers, time periods, and regions.

### Objectives

- 📈 Analyze sales trends and patterns
- 👥 Identify top-performing products and customers
- 📅 Track seasonal variations and time-based trends
- 💰 Monitor revenue and profitability metrics
- 🎯 Provide actionable insights for business strategy

## ✨ Features

- **Interactive Dashboard**: Dynamic visualizations with slicers and filters
- **Sales Performance Metrics**: Revenue, profit margins, and growth analysis
- **Product Analysis**: Best-selling products and category performance
- **Customer Segmentation**: Customer behavior and purchase patterns
- **Time-Series Analysis**: Monthly, quarterly, and yearly trends
- **Geographical Insights**: Regional sales distribution
- **Executive Summary**: High-level KPIs and recommendations

## 📊 Dataset

The project uses three main data sources:

### 1. **customers.csv**
- Customer demographic information
- Location data
- Customer segmentation attributes

### 2. **orders.csv**
- Order transaction details
- Purchase dates
- Order quantities and amounts
- Delivery information

### 3. **products.csv**
- Product catalog
- Categories and subcategories
- Pricing information
- Product attributes

## 📱 Dashboard Components

### Key Performance Indicators (KPIs)
- Total Revenue
- Total Orders
- Average Order Value
- Customer Count
- Profit Margins

### Visualizations
1. **Revenue Trend Analysis**: Line charts showing sales over time
2. **Product Performance**: Bar charts for top products and categories
3. **Customer Analytics**: Customer distribution and segmentation
4. **Regional Performance**: Geographical sales heatmaps
5. **Seasonal Patterns**: Month-over-month and quarter-over-quarter analysis

### Interactive Elements
- Date range slicers
- Product category filters
- Regional filters
- Customer segment selectors

## 📥 Installation & Usage

### Prerequisites
- Microsoft Excel 2016 or later (recommended: Excel 365)
- Basic understanding of Excel operations

### Steps to Use

1. **Clone the repository**
   ```bash
   git clone https://github.com/vinaysai7/Excel-Sales-Analysis-Dashboard.git
   cd Excel-Sales-Analysis-Dashboard
   ```

2. **Open the Dashboard**
   - Open `excel_project.xlsx` in Microsoft Excel
   - Enable macros if prompted (if applicable)

3. **Explore the Dashboard**
   - Navigate through different sheets
   - Use slicers to filter data
   - Interact with pivot tables and charts

4. **Review the Analysis**
   - Open `Sales_analysis_FNP.pdf` for detailed insights and recommendations

### Data Update (Optional)

To analyze your own data:
1. Replace the CSV files (`customers.csv`, `orders.csv`, `products.csv`) with your data
2. Ensure the column headers match the original structure
3. Refresh all pivot tables: **Data** → **Refresh All**
